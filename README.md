# Wuxi-Player

Wuxi-Player 是一款基于 Jetpack Compose 和 Media3 开发的现代 Android 音乐播放器。

## 功能特性

- **现代 UI 交互**：采用 Material 3 设计语言，支持动态色彩和沉浸式体验。
- **高性能播放**：基于 Media3 ExoPlayer，支持多种音频格式和无缝切换。
- **个性化设置**：
  - 自定义背景图片及透明度。
  - 歌词颜色与高亮实时调整。
  - 旋转封面开关。
  - 锁屏/系统媒体通知控制。
- **在线与本地**：支持在线搜索、歌词同步，以及本地音乐扫描播放。
- **文件管理**：支持自定义下载路径及通知栏快捷控制。
